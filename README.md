# Codex-Tutorials

이 저장소는 Codex를 활용한 개발 워크플로우 실습용 공간입니다.

## 문서
- [AGENTS.md](./AGENTS.md): Codex 작업 기본 규칙
- [CODEX_GUIDE.md](./CODEX_GUIDE.md): Codex 사용 가이드 (초기 세팅부터 단계별 개발 방법)

## 빠른 시작
1. 작업 목표를 한 문장으로 정의합니다.
2. `CODEX_GUIDE.md`의 0~2단계를 따라 초기 세팅을 완료합니다.
3. 3~6단계로 기능을 작은 단위로 구현하고, 각 단계마다 테스트를 검증합니다.
